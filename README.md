# My Arch dotfiles

These are my dotfiles for my current Gruvbox themed Hyprland setup on Arch, have a look around!

I manage them using [chezmoi](https://www.chezmoi.io/).

## Configurations for

* hyprland
* waybar
* wofi
* dunst
* kitty
* wlogout
* hyprlock
* Thunar
* and a bunch of other stuff
